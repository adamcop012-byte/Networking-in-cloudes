================================================================================
                        3-STEP DEPLOYMENT
================================================================================

Step 1: Fork to GitHub
  git clone <this repo>
  git remote set-url origin https://github.com/YOUR-ORG/cloud-erp-platform.git
  git push -u origin main

Step 2: Configure AWS
  cd terraform
  cp terraform.tfvars.example terraform.tfvars
  # Edit terraform.tfvars with:
  #  - repo_url: your GitHub repo
  #  - key_pair_name: your EC2 SSH key
  #  - admin_cidr: your public IP (curl https://ifconfig.me)

Step 3: Deploy
  bash deploy.sh
  # Or: cd terraform && terraform apply

That's it! Live in 30-40 minutes.
